# Basic Informational Site

## Description

This project is a simple informational website built with Node.js. It was created as part of a learning exercise to practice server-side development fundamentals, including handling HTTP requests, routing, and serving static HTML files.

The application uses a basic Node.js HTTP server to serve different pages based on the URL path. It demonstrates how a server can respond with different content depending on the requested route, including handling unknown routes with a custom 404 page.

---

## Features

- Basic Node.js HTTP server
- Routing for multiple pages:
  - Home page (`/`)
  - About page (`/about`)
  - Contact page (`/contact-me`)
- Custom 404 error page for undefined routes
- Static HTML file serving
- Simple project structure for learning purposes

---

## Tech Stack

- Node.js
- Built-in `http` module
- File system module (`fs`)
- HTML (static pages)

---

## What I Learned

- How to create a basic server using Node.js
- How routing works without frameworks like Express
- How to serve static HTML files
- How to handle 404 errors manually
- Basic file system operations in Node.js
